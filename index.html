    GakuScan requires a Google Cloud Vision API key to function, as it powers the OCR needed for text extraction.
    If want to know more, please visit the documentation.

    Steps to Obtain a Google Cloud Vision API Key

    1. Visit the Google Cloud Console.
    2. Create a new project or select an existing one.
    3. Enable the Google Cloud Vision API for your project.
    4. Set up billing (you won't be charged unless you exceed the free tier).
    5. Set up a Budget: Define a budget in the Google Cloud Console to monitor usage and avoid unexpected costs.
      Configure alerts to notify you when usage approaches your budget limit.
    6. Generate and copy your API key. Do not share your API key with anyone!
    7. Enter the API key in the settings.

    About GakuScan(学スキャン)

    GakuScan is your companion for immersive Japanese learning, making it easier to understand text from games, manga, and more. Using OCR technology and grammar highlighting, it helps you break down complex sentences while staying engaged with your favorite content.
